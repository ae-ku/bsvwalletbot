# bsvwalletbot
BitcoinSV Wallet for Telegram

How to use?

1. Register a bot with BotFather (https://t.me/botfather)

2. Copy config.example.js to config.js

3. Add MongoDB URI and Telegram Token provided by BotFather to config.js

4. Run "node index.js"
